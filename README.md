## Experiência em produção privada

Tenho trabalhado em produtos SaaS B2B com requisitos reais de produto, segurança, multi-tenancy e operação.

### SaaS multi-tenant com agentes de IA

- Frontend em **Next.js + React + TypeScript + Tailwind**
- Onboarding, dashboard, Kanban, billing e fluxos de produto
- API em **tRPC** sobre **Prisma + PostgreSQL**
- Multi-tenancy com **RLS**, views, materialized views e pgvector
- Mensageria com **Redis + BullMQ**
- Agentes com **LangGraph + Claude**, embeddings e contexto vindo do banco
- Infraestrutura com **AWS ECS, RDS, S3** e CI/CD
- Testes com **Vitest, Playwright e Testcontainers**

### Plataforma de networking, comunidades e IA generativa

- Frontend em **React + TypeScript + Vite**
- PWA, i18n, feed em tempo real e presença com WebSockets
- Backend em Edge Functions sobre Supabase Postgres
- Streaming com SSE para orquestrador de IA
- Matching semântico com embeddings para perfis, posts e oportunidades
- Integrações com Stripe, Hotmart e pipelines de vetorização
